# Servicio de Groq para Avatar

Este servicio actúa como un puente entre la GUI y la API de Groq para generar contenido viral.

## Cómo Iniciar

### Opción 1: Script Automático (Recomendada)
Haz doble clic en el archivo:
`INICIAR_GROQ.bat`

### Opción 2: Manual (PowerShell)
1. Abre una terminal en esta carpeta.
2. Ejecuta:
   ```powershell
   ..\entorno\Scripts\activate
   python app.py
   ```

## Configuración
El servicio corre por defecto en el **puerto 5005**.
Asegúrate de tener tu API Key configurada en `app.py`.
